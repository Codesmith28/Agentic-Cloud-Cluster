Problem: "Worker not found"
Solution: Register the worker first using the register command

Problem: "Worker is not active"
Solution: Check that the worker is running and sending heartbeats
         Wait a few seconds after registration

Problem: "Master not registered yet"
Solution: Wait for master to register with worker (automatic)
         Or restart the worker after master is running

Problem: "Failed to pull image"
Solution: Check Docker is running on worker machine
         Verify image name is correct
         Check network connectivity

Problem: "Cannot connect to worker"
Solution: Verify worker IP and port are correct
         Check firewall settings
         Ensure worker is actually running
